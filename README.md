# Cardiovascular Disease Risk Analysis and Prediction

## Project Overview

This project analyzes cardiovascular disease risk factors using data analytics and machine learning techniques.

## Dataset

Source: Kaggle Cardiovascular Disease Dataset

Features:
- Age
- Gender
- Height
- Weight
- Blood Pressure
- Cholesterol
- Glucose
- Smoking
- Alcohol Consumption
- Physical Activity

Target:
- Cardiovascular Disease

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Project Workflow

1. Data Cleaning
2. Feature Engineering
3. Exploratory Data Analysis
4. Correlation Analysis
5. Visualization
6. Machine Learning Prediction

## Key Findings

- Cardiovascular disease prevalence increases with age.
- High blood pressure is strongly associated with disease occurrence.
- Elevated cholesterol levels increase disease risk.
- Higher BMI is associated with cardiovascular disease.
- Elevated glucose levels contribute to disease prevalence.

## Machine Learning Results

Logistic Regression was used for prediction and evaluated using Accuracy Score, Confusion Matrix, and Classification Report.
